Модификация популярного плеера **SmartTube**, возвращающая эстетику оригинального приложения YouTube с улучшенным современным интерфейсом.

---

## ✨ Основные изменения

### 🎨 Визуальная часть
* **Icon Pack:** Все иконки заменены на оригинальные (Stock YouTube).
* **Branding:** Название приложения изменено на **YouTube**.
* **Adaptive Menu:** Цвет иконки меню автоматически подстраивается под выбранную тему оформления.
* **Modern UI:** Окна превью и строка поиска получили эстетичное закругление.
* **Rounded Interface:** Иконки внутри интерфейса отредактированы и приведены к овальной форме.

### ⚙️ Техническая часть
* **Clean Mod:** Образ SmartTube остался нетронутым — стабильность оригинала сохранена.
* **No Conflicts:** Мод устанавливается **рядом** с официальным SmartTube (не заменяет его).
* **Universal:** Используется универсальная архитектура (подходит для большинства устройств).
* **Easy Migration:** Имя пакета (Package Name) оставлено прежним — все ваши настройки подхватятся автоматически.

## 🚀 Установка

1. Перейдите в раздел **[Releases](https://github.com/ArmanychKZ/Smarttube-original-icon-mod/releases)**.
2. Скачайте актуальный APK-файл.
3. Разрешите установку из неизвестных источников на вашем Android TV.
4. Установите и наслаждайтесь привычным видом YouTube!
